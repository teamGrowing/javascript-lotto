# 기능 목록

## 주요 기능

-[ ] 구입 금액 입력받기  
-[ ] 로또 번호 생성하기  
-[ ] 구매 로또 번호 출력 하기  
-[ ] 당첨 번호 입력받기  
-[ ] 보너스 번호 입력받기  
-[ ] 당첨 통계 및 수익률 출력하기

## 예외 처리

### 금액 예외

-[ ] 숫자가 아닌 경우  
-[ ] 1000원 단위가 아닌 경우

### 당첨번호 예외

-[ ] 숫자가 아닌 경우  
-[ ] 범위에서 벗어난 경우  
-[ ] 중복된 숫자인 경우  
-[ ] 6개가 아닌 경우  
-[ ] 쉼표로 구분하여 입력하지 앟은 경우

### 보너스 번호 예외

-[ ] 숫자가 아닌 경우  
-[ ] 중복된 숫자인 경우  
-[ ] 범위에서 벗어난 경우
